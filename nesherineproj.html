Nesherine (1st Unity Game, Unpublished)

This was my very 1st Completed game on Unity, this was created in 2020 and was inspired by the game Catherine. Catherine is a block-moving puzzle solver, in which the goal for that game is to make it to the top by creating a path.
I wanted to replicate that mechanic of being able to move cubes around, it was a bit tricky to make it work but it eventually worked! Never published this one as I thought there were many other ways to improve the game but never really had the time.
