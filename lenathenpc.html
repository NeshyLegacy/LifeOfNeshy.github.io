LenaTheNPC

I developed an interactive Twitch Bot called "LenaTheNPC" with the help of Kruiz Control, which serves as a valuable addition to my portfolio. The bot's programming language primarily consists of Pseudocode, although it incorporates some elements of JavaScript and allows for API Calling and Discord Webhooks integration.

LenaTheNPC was created with the aim of enhancing viewer engagement on my Twitch Channel by offering a range of features such as entertaining minigames and a "Question of the Day". It also help my channel stand out on the platform with things like delivering unique welcome messages and offering special commands.
