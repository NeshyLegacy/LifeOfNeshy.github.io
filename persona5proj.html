Persona 5 Fan-Made App

This project was created in 2020 with AIT under the subject "Interface Experience and Design", where we basically had to make an app. Going through all the stages like Brainstorming, rough drafts and sketches and more. We had to create powerpoint presentation in each version of the app, gather feedback and use that feedback to improve the app.
The rough draft and sketeches were made by using the Marvel App, then the Final version of the App was created in Adobe XD which we had to self learn in a very short time.
